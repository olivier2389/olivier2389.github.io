Puntos de conexión

📦
Agregar nueva plantilla → js/templateRegistry.js Agrega un objeto {id, icon, nombre, desc, estructura} al registro. Se renderiza automáticamente, sin tocar nada más.
🗺️
Agregar nueva materia → js/app.js (MATERIAS_CONFIG) Genera el JSON con el extractor Python, colócalo en data/, y descomenta la entrada en MATERIAS_CONFIG. Aparece en el selector del header.
🤖
Conectar a API de IA → js/app.js (sendToAPI) El código está escrito y comentado. Solo descomenta, agrega tu API key y cambia el botón "Generar" por "Generar y enviar".
💾
Persistencia → js/app.js (EduStorage) Código listo y comentado. Descomenta EduStorage para guardar selecciones y prompts entre sesiones del navegador.
